"# StockAlert" 


### Developer Instructions

1) To instal `node_modules`
    ```BASH
    npm install
    ```
1) For compiling TypeScript files
    ```BASH
    npx tsc -w
    ```
    > Note: Leave this terminal window open
1) To run the server
    ```BASH
    node ./dist/main.js
    ```
